gene 問診チェックアプリ PWA版

【内容】
・問診票 2ページ目をそのまま反映
・12カテゴリ
・82項目
・タップ式チェック
・自動集計
・軽度 / 中度 / 重度の自動判定
・カテゴリ別グラフ
・ホーム画面追加対応
・オフライン対応

【使い方】
1. zipを解凍
2. 中身をGitHub Pagesなどにそのままアップロード
3. 公開URLをChromeで開く
4. ホーム画面に追加

【注意】
・zipのままアップロードしない
・index.html が最上層にある状態でアップロードする
・PWAは content:// や file:// ではなく https:// で開く
