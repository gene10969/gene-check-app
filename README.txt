gene 問診チェックアプリ 保存・比較機能付きPWA版

【追加した機能】
・患者ごと保存
・保存済み患者一覧
・患者検索
・前回比較
・保存履歴読み込み
・JSONバックアップ書き出し
・JSONバックアップ読み込み
・今回データ削除

【今回反映】
・問診票2ページ目をそのまま反映
・12カテゴリ
・82項目

【使い方】
1. zipを解凍
2. 中身をGitHub Pagesなどにアップロード
3. 公開URLをChromeで開く
4. ホーム画面に追加
5. 患者名入力 → 保存
6. 次回来院時は保存済み患者から選択 → 前回比較

【保存について】
・通常保存はブラウザ内保存です
・端末変更やブラウザ初期化に備えて、定期的にJSON書き出し推奨
